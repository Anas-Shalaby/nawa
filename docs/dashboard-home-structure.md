# شكل صفحة الداشبورد (Mission Control)

المسار: `/{locale}/dashboard`  
الملف: `src/components/dashboard/DashboardShell.tsx`

**غرفة عمليات الاستقبال** — شبكة كثيفة من 3 ألواح بدون تمرير للصفحة الأم.

```
lg:grid-cols-12 · h-[calc(100vh-80px)] · overflow داخلي لكل لوح
┌──────────────┬────────────────────────┬──────────────┐
│ عمليات سريعة │ الساحة الحية (Kanban)  │ رادار العيادة│
│ col-span-3   │ col-span-6             │ col-span-3   │
│ Walk-in مدمج │ بالخارج|انتظار|طبيب    │ غرف + خزنة  │
│ + مهام عاجلة │ أو قائمة يومية فارغة   │ + إشغال %   │
└──────────────┴────────────────────────┴──────────────┘
```

RTL: اللوح الأول يظهر يميناً.

البيانات من `fetchTodayAppointments()` تشمل: مواعيد، مدفوعات اليوم، متأخرات أمس، غرف مشتقة من الجلسات، نسبة الإشغال.
